# 零售运营数据指标模型原型

此仓库为零售运营数据指标模型原型设计。

这是一个独立的静态原型页面，不影响同目录下之前的原型文件。

## 本地运行

标准更新流程建议使用本地静态服务访问，这样修改 `src/` 下文件后，直接刷新页面即可看到最新原型。

```bash
python -m http.server 4173
```

默认访问地址：

- 本地开发：`http://127.0.0.1:4173/`
- 发布后默认地址：`https://<你的用户名>.github.io/<仓库名>/`

## 发布到 GitHub Pages

1. 新建 GitHub 仓库并把当前目录推送到 `main`
2. 在仓库设置中启用 **Pages / GitHub Actions**
3. 推送后自动执行 `.github/workflows/deploy.yml`
4. 发布地址格式通常为：

```text
https://<你的用户名>.github.io/<仓库名>/
```

后续更新只需要修改当前目录下的 `src/` 文件并推送到 `main`，然后刷新 GitHub Pages 地址即可看到最新版本。

## 独立性说明

- 现有的 `原型脚本.ipynb` 未被修改
- 当前页面入口为 `index.html`
- 原型核心代码位于 `src/`
